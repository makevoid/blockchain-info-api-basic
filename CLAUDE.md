## Commands

```bash
npm test          # run all tests (hits live blockchain.info API)
npm run build     # bundle for browser via browserify (outputs bundle.js)
```

Run a single test by name:
```bash
npx jest --testNamePattern="balance"
```

## Architecture

This is a minimal Node.js wrapper around the [blockchain.info](https://blockchain.info) public API, also bundled for browser use via browserify.

**`index.js`** — the entire library. Exports three functions:
- `balance(address)` — fetches satoshi balance via `/q/addressbalance/`
- `utxos(address)` — fetches unspent transaction outputs via `/unspent`
- `pushTx(txHex)` — broadcasts a raw transaction via `/pushtx`

**`bundle.js`** — browserify output, exposes the library as `BchainApi` global. Generated by `build.sh`; not edited directly.

**Tests** (`index.test.js`) hit the live API against a known address with a fixed balance. `pushTx` test is skipped (`xtest`) because it would require mocking.

The library is isomorphic: `isomorphic-fetch` polyfills `fetch` for Node; in browsers the native `fetch` is used.
